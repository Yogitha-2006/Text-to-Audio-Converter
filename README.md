🎤 Text-to-Audio Converter

A simple and interactive Text-to-Audio Converter built with Python, Streamlit, and Edge-TTS. This application converts text into natural-sounding speech and allows users to play or download the generated audio.

🚀 Features
Convert text into speech
Multiple AI voice options
Simple Streamlit interface
Play generated audio in the browser
Download audio as an MP3 file
🛠️ Technologies Used
Python
Streamlit
Edge-TTS
